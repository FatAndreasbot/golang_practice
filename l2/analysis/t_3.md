```go
package main

import "fmt"

type CustomError struct {
	message string
}

func (e *CustomError) Error() string {
	return e.message
}

func returnError(flag bool) error {
	if flag {
		return &CustomError{"Что-то пошло не так"}
	}
	var err *CustomError
	return err
}

func main() {
	err1 := returnError(true)
	err2 := returnError(false)

	fmt.Println("err1 == nil:", err1 == nil)
	fmt.Println("err2 == nil:", err2 == nil)

}
```
## Вывод
```
err1 == nil: false
err2 == nil: false
```
## Объяснение
- Для того чтобы значение интерфейса считалось `nil` необходимо
    1. `tab *itab` был `nil`
    2. `data unsafe.Pointer` был `nil`

1. Выводится указатеьл на структуру
    И `data` и `tab` не равны `nil`
2. `tab` не равен `nil`
