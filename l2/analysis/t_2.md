```go
package main

import (
	"fmt"
)

type errorString struct {
	s string
}

func (e errorString) Error() string {
	return e.s
}

func checkErr(err error) {
	fmt.Println(err == nil)
}

func main() {
	var e1 error
	checkErr(e1)

	var e *errorString
	checkErr(e)

	e = &errorString{}
	checkErr(e)

	e = nil
	checkErr(e)
}
```
## Вывод
```
true
false
false
false
```
## Объяснение
- Для того чтобы значение интерфейса считалось `nil` необходимо
    1. `tab *itab` был `nil`
    2. `data unsafe.Pointer` был `nil`

1. В пустой переменной типа интерфейса - оба значения `nil`
2. В данном случае `tab *itab` != `nil` 
3. И `tab *itab` != `nil`, и `tab *itab` != `nil` 
4. Идентично случаю 2
