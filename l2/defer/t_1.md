```go
package main

import (
	"fmt"
)

func main() {
	fmt.Println("start")
	for i := 1; i < 4; i++ {
		defer fmt.Println(i)
	}
	fmt.Println("end")
}
```
## Вывод
```
start
end
3
2
1
```
## Объяснение

`defer` произходит в конце функции, перед возвратом значения
Исполнение функций в defer идет по принципу `LIFO`
