```go
package main

import "fmt"

func main() {
	value := 123
	defer fmt.Println(value)
	changeValue(&value)
}
func changeValue(value *int) {
	*value = 456
}
```
## Вывод
```
123
```
## Объяснение
Исполнение (если так можно сказать) происходит в
моменте вызова `defer`, а не в конце метода, где
он находится
## Вариант исправления
```go
func main() {
   	value := 123
	changeValue(&value)
	defer fmt.Println(value)
}
```
